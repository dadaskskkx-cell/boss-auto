# Boss 客户端试用说明

本项目当前只支持 `Boss 直聘 PC 客户端`，不支持网页端。

## 使用前准备

1. 安装 `Boss 直聘 PC 客户端`
2. 安装 `Python 3.12`
3. 打开客户端并登录账号

## 启动步骤

双击 `launch.command`。

首次启动会自动创建 `venv` 并安装依赖。

如果本地还没有 `config/config.yaml`，启动器会先按模板生成一份。
这时需要先把里面的 `api_key`、`vision_api_key` 改成你自己的，再重新双击启动。

启动后会先进入“岗位 JD 输入向导”：

1. 输入本次岗位名称
2. 粘贴本次岗位 JD
3. 输入完成后单独一行输入 `/done`
4. 检查自动生成的关键词和硬筛规则
5. 确认后脚本才会开始执行

## 当前脚本能力

- 识别 Boss 客户端窗口
- 读取当前推荐列表候选人卡片
- 根据本次输入的岗位 JD 自动生成筛选规则
- 对通过筛选的候选人点击 `打招呼`
- 自动下滑继续扫描更多候选人

## 注意事项

- 当前方案是 `客户端 RPA`，不是网页自动化
- 如果页面弹出 `账号状态异常`、`暂不支持此操作`、`去充值`、`搜索畅聊卡`，这是 Boss 侧限制，不是脚本报错
- 每次启动都会重新输入岗位 JD，并覆盖 `config/profile.yaml`

## 关键配置文件

- `config/profile.yaml`：岗位 JD、筛选规则
- `config/config.yaml`：运行配置、LLM 配置
- `config/config.yaml.example`：初始化模板
